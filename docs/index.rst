CellProfiler
============

CellProfiler is a free open-source software for quantitative analysis of biological images. We demonstrate how to integrate CellProfiler and OMERO using the CellProfiler Python API and the OMERO Python API.

For more details, visit the `CellProfiler website <https://cellprofiler.org/>`_.

Changes to the documentation or the materials should be made directly in the `omero-guide-cellprofiler repository <https://github.com/ome/omero-guide-cellprofiler>`_.
